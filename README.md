JHA2 Calculator

A premium Android calculator built with determination, GitHub Actions, AI assistance, questionable debugging decisions, and an unhealthy amount of persistence.

Why "JHA2"?

Named after a legendary individual known throughout the group for:

- Being approximately three software updates tall.
- Maintaining hair length that suggests either rockstar ambitions or a long-running disagreement with barbers.
- Declaring future plans of becoming a Software Engineer before discovering what Gradle errors look like.
- Achieving local fame as the group's official Activa racing specialist.
- Possessing the confidence to explain technology while simultaneously being confused by technology.

This calculator is a tribute to that remarkable combination of ambition, optimism, and mechanical transportation.

Features

- Basic calculations
- Scientific functions
- Premium UI
- History panel
- Settings panel
- Enough debugging history to qualify as character development

Development Story

The app was built through a highly scientific process:

1. Generate code.
2. Get an error.
3. Fix error.
4. Generate a new error.
5. Repeat until APK appears.

Current success rate exceeds expectations.

Performance

- Faster than JHA2's Activa on uphill roads.
- More reliable than group project deadlines.
- Better at calculations than estimating one's own height.
